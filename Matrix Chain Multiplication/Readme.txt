Programming Language used: Python with Tkinter package for GUI

1. MatChainMul.py is the main python file, which takes the inputs from the file input.txt

2. The values of the matrices are generated randomly between 0 and 10, based on the dimensions provided in input.txt file.

3. customize.py is a python file which can be configured to change color & font styles of objects in the target program.

4. To slow down the animation, increase the value of transition_time in MatChainMul.py
